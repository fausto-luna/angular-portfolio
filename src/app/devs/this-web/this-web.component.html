Angular portfolio v1.0

This website was meant to be an online portfolio where I can showcase my work as an artist and web developer.

At the same time, this particular web application is the result of the Front-end Web Development training program I have been enrolled in since November 2019 until July 2020 at IT Academy of Barcelona Activa.

Web desktop navigation demo

The seed

As soon as I finished my photography studies back in 2008 I realised that displaying my work on the web was highly important, so immediately after graduating, being able to build my own website from scratch was kind of a dream. However, since the first time I engaged with many of all the online portfolio templates out there knew that templates are great solutions, but for some reason I did not have enough. I wanted to know what I was doing AND do it limitless. Well, looks like the time has come.

Angular + Bootstrap

I fully developed this mobile and desktop web app with Angular, the worldwide spread framework developed by Google. why Angular? Basically, it is a combination of HTML and TypeScript. Also @ng-bootstrap (Angular powered Bootstrap) was installed in order to optimise the user interface styling.

Responsive and mobile-first

Angular Material is installed in this app in order to build a consistent and modern UI component infrastructure, providing highly useful utilities for the development of responsive layouts that react to screen-size changes, having as result a dynamic UI that works across mobile, web and desktop.

Bootstrap responsive layout is also implemented by the specification of its media query ranges —or breakpoints— in Bootstrap's source Sass files. However, additional CSS media queries were implemented for behavior customisation.

iPhone X navigation demo

Animations

It was important for me to add Angular animations to this project. Nothing too flashy, just very simple slide-in/slide-out transitions between pages that perform the illusion of motion, making a significant improvement to the app as well as providing an accesible, relevant and meaningful user experience (UX).
