Cuck Norris Jokes

Carlos Ray "Chuck" Norris

You know Chuck Norris is unique, incomparable right? Well, so are the jokes about him and there are plenty of them!

With this little app you can access to an almost infinite number of jokes where our hero of heroes is the main character, of course! Your friends are going to be blown up by all the resources you are going to deploy at every party. Note that many of the jokes you are going to get are IT-oriented.

Ready for starting up your career as a comedian?

JQuery & AJAX

In this tiny project I implemented an AJAX GET call with JQuery, receiving results from a REST API endpoint. In other words, the application makes a request looking for read-only data located in a remote server. In this case, it access to a JSON (JavaScript Object Notation) file where jokes are stored.

This web app is the result of an exercise for the Front-end training program I have been enrolled in at IT Academy of Barcelona Activa.
