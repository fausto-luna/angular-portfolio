Vehicle Registration OOP

CRUD application

The mission for this project is to implement real-world web application able to perform all the tasks needed in a user registration process. For example, a website for a car workshop where users can register their vehicles. In other words, this approach implements a persistent storage application: create, read, update and delete (CRUD).

OOP [HTML5 + CSS3 + TypeScript]

This is an Object-oriented programming (OOP) application developed with HTML5, CSS3 and TypeScript [ECMAScript 6] during the Front-end Web Development training program I have been enrolled in at IT Academy of Barcelona Activa. The goal for this exercise was to implement the basic concepts of OOP; encapsulation, abstraction, inheritance and  polymorphism.
